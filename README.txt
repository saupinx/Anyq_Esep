ANYQ ESEP — ОБНОВЛЕННЫЙ ПРОЕКТ

Структура проекта:

1) index.html
   Главная HTML-разметка сайта.
   Здесь находятся все секции: шапка, главный экран, услуги, процесс, полезная информация, контакты.

2) css/style.css
   Все стили сайта.
   Отвечает за:
   - цвета (серый / зеленый / красный)
   - сетку и отступы
   - адаптивность под телефон / планшет / компьютер
   - карточки, кнопки, секции, формы

3) js/main.js
   JavaScript-логика сайта.
   Отвечает за:
   - мобильное меню
   - подсветку активного пункта меню при прокрутке
   - демо-отправку формы

4) images/
   Папка для логотипа, фотографий, иконок и любых изображений.
   Сейчас оставлена пустой на будущее.
